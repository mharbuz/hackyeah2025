TRASY:
------
GET  /symulacja-emerytury          → pension-simulation.index
POST /api/pension/simulate          → pension-simulation.simulate

URUCHOMIENIE:
-------------
1. php artisan route:cache
2. npm run build (lub npm run dev)
3. php artisan serve
4. Odwiedź: http://localhost:8000/symulacja-emerytury

UWAGA:
------
Jeśli wystąpią problemy z uprawnieniami przy generowaniu tras Wayfinder:
sudo chown -R $USER:$USER resources/js/
php artisan wayfinder:generate
